# Todo

WPF приложение.

## Начало работы

Эти инструкции помогут вам запустить проект на локальном компьютере для разработки и тестирования.

### Необходимые условия

Что нужно установить/использовать перед началом работы:
- Visual Studio 2022
- Git

### Установка

Пошаговая инструкция по запуску проекта:

1. Откройте Visual Studio
2. Нажмите на "Клонировать репозиторий"
3. Вставьте ссылку на репозиторий: https://github.com/kil0what/todo
4. После клонирования перейдите на ветку "lab1"
5. Запустите проект нажатием F5
